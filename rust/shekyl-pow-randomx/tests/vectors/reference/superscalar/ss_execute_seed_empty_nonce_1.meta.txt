SuperscalarHash execution vector — ss_execute_seed_empty_nonce_1.bin
====================================================================

Spec attestation
----------------
Cross-checks shekyl-pow-randomx's `superscalar::execute_superscalar`
(spec §7.3 step 5) against the v2 RandomX fork's
`randomx::executeSuperscalar`. Layer B vector paired with Layer A
program `ss_program_seed_empty_nonce_1.bin` (the nonce-mixing
isolation program). See `ss_execute_seed_empty_nonce_0.meta.txt`
for the Layer B vector rationale; this vector exercises the same
execution path against a different generated program.

Inputs (hardcoded in `_generator/gen.cpp`)
------------------------------------------
program = Layer A vector `ss_program_seed_empty_nonce_1.bin`
input_r = [0, 1, 2, 3, 4, 5, 6, 7]  — same as exec_v1 by intent

Output layout
-------------
64 bytes = `r[8]` after `executeSuperscalar(r, prog)`, each register
written as little-endian `u64`.
