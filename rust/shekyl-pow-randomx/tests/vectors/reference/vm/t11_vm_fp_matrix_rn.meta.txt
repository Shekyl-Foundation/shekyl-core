Spec attestation
----------------

the v2 RandomX fork's FP dispatch under MXCSR mode 0 (round-to-
nearest) across the full 9-opcode FP set:

  FSWAP_R (120), FADD_R (124), FADD_M (140), FSUB_R (145),
  FSUB_M (161), FSCAL_R (166), FMUL_R (172), FDIV_M (204),
  FSQRT_R (208).

slice of the T11-T14 FP rounding-mode matrix.
