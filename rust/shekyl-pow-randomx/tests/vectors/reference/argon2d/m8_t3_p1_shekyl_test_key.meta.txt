Argon2d derived test vector — m8_t3_p1_shekyl_test_key.bin
==========================================================

Spec attestation
----------------
Cross-checks shekyl-pow-randomx's `argon2d::fill_cache` against the
v2 RandomX fork's reference Argon2 implementation at the boundary
case `m_cost = MIN_M_COST = 2 * SYNC_POINTS = 8`. Per
`external/randomx-v2/doc/specs.md` §7.1 the Cache derivation is an
omit-finalizer Argon2d fill; this vector exercises that surface at
the smallest legal m_cost.

----------
algorithm   = Argon2d (discriminant 0)
version     = 0x13 (ARGON2_VERSION_13)
m_cost      = 8        (1 KiB blocks; total 8 KiB output)
t_cost      = 3        (matches RANDOMX_ARGON_ITERATIONS)
p_cost      = 1        (matches RANDOMX_ARGON_LANES)

key         = "Shekyl RandomX v2 Phase 2a test key"
output_len  = 0        (omit finalizer; output is the filled memory)

Generator

source      = ./_generator/gen.c
